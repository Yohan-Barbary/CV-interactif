Lexique Pratique

Guide des commandes essentielles pour Docker et Git

Docker

Gestion des Images

docker pull [image]

Télécharger une image depuis Docker Hub

docker images

Lister toutes les images disponibles

docker build -t [nom] .

Construire une image à partir d'un Dockerfile

Création de Conteneurs

docker run -d --name [nom] [image]

Créer et démarrer un conteneur en arrière-plan

docker run -it [image] bash

Créer et démarrer un conteneur en mode interactif

docker run -p 8080:80 [image]

Créer un conteneur avec redirection de port (local:conteneur)

docker run -e VARIABLE=valeur [image]

Créer un conteneur avec des variables d'environnement

docker run --restart always [image]

Créer un conteneur qui redémarre automatiquement

docker run --memory="512m" [image]

Créer un conteneur avec limite de mémoire

Gestion des Conteneurs

docker ps

Lister les conteneurs en cours d'exécution

docker stop [conteneur]

Arrêter un conteneur

Docker Compose

docker-compose up

Démarrer tous les services définis

docker-compose down

Arrêter tous les services

Gestion des Volumes

docker volume create [nom]

Créer un nouveau volume

docker volume ls

Lister tous les volumes

docker run -v [volume]:[chemin] [image]

Monter un volume dans un conteneur

Réseau Docker

docker network create [nom]

Créer un nouveau réseau

docker network connect [réseau] [conteneur]

Connecter un conteneur à un réseau

docker network ls

Lister tous les réseaux

Git

Configuration

git config --global user.name "[nom]"

Configurer le nom d'utilisateur

git config --global user.email "[email]"

Configurer l'email

Commandes de Base

git init

Initialiser un nouveau dépôt

git add [fichier]

Ajouter des fichiers à l'index

git commit -m "[message]"

Créer un commit avec un message

Synchronisation avec GitHub

git clone [url]

Cloner un dépôt distant sur votre machine

git push

Envoyer vos modifications vers le dépôt distant

git pull

Récupérer les modifications du dépôt distant

git fetch

Récupérer les modifications sans les fusionner

git push -f origin [branche]

Forcer l'envoi des modifications (à utiliser avec précaution)

git pull --rebase origin [branche]

Récupérer et réorganiser l'historique local

Gestion des Branches

git branch [nom]

Créer une nouvelle branche

git checkout [branche]

Changer de branche

git merge [branche]

Fusionner une branche dans la branche courante

Gestion des Remotes

git remote add origin [url]

Ajouter un dépôt distant

git push -u origin [branche]

Pousser une branche vers le dépôt distant

git pull origin [branche]

Récupérer et fusionner les changements distants

Gestion des Modifications

git status

Voir l'état des fichiers

git diff

Voir les modifications non indexées

git log

Voir l'historique des commits

git reset --hard HEAD^

Annuler le dernier commit
