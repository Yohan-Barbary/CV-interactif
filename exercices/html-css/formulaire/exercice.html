Objectif

Créer un formulaire d'inscription interactif avec validation en temps réel et retours visuels.

Consignes

Structure requise

  • Champs pour nom, email et mot de passe
  • Messages de validation personnalisés
  • Indicateur de force du mot de passe
  • Bouton pour afficher/masquer le mot de passe

Exigences techniques

  • Validation en temps réel avec JavaScript
  • Styles personnalisés pour les états des champs :
    • Normal
    • Focus
    • Erreur
    • Validé
  • Animations pour les messages d'erreur

Bonus

  • Ajouter une vérification de force du mot de passe
  • Implémenter une confirmation du mot de passe
  • Ajouter des transitions fluides

Ressources

  • Guide MDN sur les formulaires
  • API de validation des contraintes
  • Animations CSS
