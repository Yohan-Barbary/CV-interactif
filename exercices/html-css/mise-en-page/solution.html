    <section class="hero-section">
        <h2>Section Héro</h2>

CSS


/* Variables CSS */
:root {
    --primary-color: #4CAF50;
    --text-color: #333;
    --background-color: #f5f5f5;
}

/* Header avec Flexbox */
.site-header {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
}

/* Grille responsive */
.cards-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 2rem;
    padding: 2rem;
}

/* Media Queries */
@media (max-width: 768px) {
    .cards-grid {
        grid-template-columns: 1fr;
    }
}

Explications

1. Structure Flexbox

Pour l'en-tête, nous utilisons Flexbox pour :

  • Aligner les éléments horizontalement avec justify-content
  • Centrer verticalement avec align-items
  • Permettre une disposition flexible sur mobile

2. Grille CSS

La grille de cartes utilise :

  • grid-template-columns avec auto-fit pour une adaptation automatique
  • minmax pour définir une taille minimum et maximum des colonnes
  • gap pour l'espacement uniforme
