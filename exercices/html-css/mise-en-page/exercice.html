Objectif

Créer une mise en page responsive qui s'adapte à différentes tailles d'écran en utilisant Flexbox et Grid.

Consignes

Structure requise

  • Un en-tête (header) avec logo et menu de navigation
  • Une section héro avec image de fond et texte superposé
  • Une grille de 6 cartes de contenu
  • Un pied de page avec 3 colonnes

Exigences techniques

  • Utiliser Flexbox pour la navigation
  • Utiliser Grid pour la grille de cartes
  • Breakpoints :
    • Mobile : < 768px (1 colonne)
    • Tablette : 768px - 1024px (2 colonnes)
    • Desktop : > 1024px (3 colonnes)

Bonus

  • Ajouter des animations au survol des cartes
  • Implémenter un menu burger pour mobile
  • Utiliser des variables CSS pour les couleurs
