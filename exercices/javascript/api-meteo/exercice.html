API Météo

Avancé

Objectif

Créez une application météo qui utilise l'API OpenWeatherMap pour afficher les conditions météorologiques actuelles et les prévisions.

Fonctionnalités requises

  • Recherche de ville avec autocomplétion
  • Affichage des conditions météo actuelles
  • Affichage des prévisions sur 5 jours
  • Gestion des erreurs et états de chargement
  • Stockage des dernières recherches

Exigences techniques

  • Utilisation de l'API Fetch et des Promesses
  • Gestion asynchrone avec async/await
  • Manipulation dynamique du DOM
  • Gestion du localStorage
  • Gestion des erreurs try/catch

Bonus

  • Ajout d'animations pour les transitions
  • Affichage d'une carte interactive
  • Mode sombre/clair
  • Support multilingue
