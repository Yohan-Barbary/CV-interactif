Objectif

Créez une application de liste de tâches interactive avec stockage local des données.

Fonctionnalités requises

  • Ajouter une nouvelle tâche
  • Marquer une tâche comme terminée
  • Supprimer une tâche
  • Sauvegarder les tâches dans le localStorage

Exigences techniques

  • Utiliser JavaScript vanilla (pas de framework)
  • Implémenter la persistance des données avec localStorage
  • Gérer les événements utilisateur
  • Manipuler le DOM de manière dynamique

Bonus

  • Ajouter une fonction de filtrage (Toutes/Actives/Terminées)
  • Permettre la modification des tâches existantes
  • Ajouter des animations CSS
