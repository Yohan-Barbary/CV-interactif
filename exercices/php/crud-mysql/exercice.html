CRUD MySQL

Intermédiaire

Note: Cette démo est actuellement simulée en JavaScript car GitHub Pages ne prend pas en charge PHP. Dans un environnement serveur réel, les opérations seraient effectuées via PHP et MySQL.

Description

Dans cet exercice, vous allez créer une application CRUD (Create, Read, Update, Delete) complète avec PHP et MySQL. L'application permettra de gérer une liste de produits avec leurs caractéristiques.

Objectifs

  • Créer une base de données MySQL
  • Implémenter les opérations CRUD
  • Gérer les formulaires et la validation des données
  • Utiliser PDO pour la connexion à la base de données
  • Gérer les erreurs et les messages de confirmation

Instructions

  1. Créez une base de données 'products_db' avec une table 'products'
  2. Implémentez un formulaire pour ajouter des produits
  3. Affichez la liste des produits avec options de modification et suppression
  4. Ajoutez la validation des données côté serveur
  5. Gérez les messages de succès et d'erreur
  6. Implémentez la pagination des résultats
