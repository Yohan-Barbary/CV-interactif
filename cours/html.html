<HTML5 - Les Fondamentaux/>

<Introduction à HTML5>

HTML (HyperText Markup Language) est le langage standard pour créer et structurer des pages web.

Caractéristiques :

  • Langage de balisage
  • Structure sémantique
  • Compatible multiplateforme
  • Base du web moderne

<Structure de base>

Éléments essentiels :

  • <!DOCTYPE html> : Déclare le type de document
  • <html> : Élément racine
  • <head> : Métadonnées et liens
  • <body> : Contenu visible

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon site web</title>
</head>
<body>
    <h1>Bienvenue sur mon site</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>

<Balises essentielles>

Balises de texte :

  • <h1> à <h6> : Titres
  • <p> : Paragraphe
  • <strong> : Texte important
  • <em> : Texte emphase
  • <br> : Saut de ligne
  • <hr> : Ligne horizontale

<h1>Titre principal</h1>
<h2>Sous-titre</h2>

<p>
    Ceci est un paragraphe avec du texte 
    <strong>important</strong> et 
    <em>mis en emphase</em>.
</p>

<p>Premier paragraphe</p>
<hr>
<p>Second paragraphe</p>

Balises de structure :

  • <div> : Conteneur générique
  • <span> : Conteneur en ligne
  • <header> : En-tête
  • <nav> : Navigation
  • <main> : Contenu principal
  • <footer> : Pied de page

<Attributs HTML>

Attributs communs :

  • class : Classes CSS
  • id : Identifiant unique
  • style : Styles CSS en ligne
  • title : Info-bulle

<div class="container" id="main">
    <p style="color: blue;">Texte bleu</p>
    <img src="image.jpg" alt="Description">
    <a href="https://exemple.com" title="Visiter le site">
        Lien
    </a>
</div>

<HTML Sémantique>

Balises sémantiques HTML5 :

  • <article> : Contenu autonome
  • <section> : Section thématique
  • <aside> : Contenu connexe
  • <figure> : Contenu autonome référencé
  • <figcaption> : Légende de figure

<article>
    <header>
        <h1>Titre de l'article</h1>
    </header>

    <section>
        <h2>Section 1</h2>
        <p>Contenu...</p>
    </section>

    <aside>
        <h3>À propos</h3>
        <p>Information complémentaire</p>
    </aside>

    <figure>
        <img src="image.jpg" alt="Description">
        <figcaption>Légende de l'image</figcaption>
    </figure>
</article>

<Formulaires>

Éléments de formulaire :

  • <form> : Conteneur de formulaire
  • <input> : Champ de saisie
  • <label> : Étiquette
  • <select> : Liste déroulante
  • <textarea> : Zone de texte
  • <button> : Bouton

<form action="/submit" method="post">
    <div>
        <label for="nom">Nom :</label>
        <input type="text" id="nom" name="nom" required>
    </div>

    <div>
        <label for="email">Email :</label>
        <input type="email" id="email" name="email">
    </div>

    <div>
        <label for="message">Message :</label>
        <textarea id="message" name="message"></textarea>
    </div>

    <button type="submit">Envoyer</button>
</form>

<Multimédia>

Éléments multimédia :

  • <img> : Images
  • <video> : Vidéos
  • <audio> : Sons
  • <iframe> : Contenu embarqué

<!-- Image responsive -->
<img src="image.jpg" 
     alt="Description"
     width="800" 
     height="600">

<!-- Vidéo avec contrôles -->
<video controls width="100%">
    <source src="video.mp4" type="video/mp4">
    Votre navigateur ne supporte pas la vidéo.
</video>

<!-- Audio avec contrôles -->
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Votre navigateur ne supporte pas l'audio.
</audio>

<!-- Iframe pour contenu embarqué -->
<iframe src="https://www.youtube.com/embed/video"
        width="560" 
        height="315"
        allowfullscreen>
</iframe>
